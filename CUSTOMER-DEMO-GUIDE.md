# Turbo Asset - Customer-Ready IWMS Platform

## 🚀 Quick Start Guide

### Prerequisites
- Node.js 18.0.0 or higher
- npm or yarn package manager

### Running the Complete System

#### 1. Start Backend API Server
```bash
# From the root directory
node demo-server.js
```

Backend will be available at: **http://localhost:3001**

API Endpoints:
- `GET /health` - Server health check
- `GET /api/assets` - Asset management  
- `GET /api/assets/stats` - Asset statistics
- `GET /api/work-orders` - Work order management
- `GET /api/work-orders/stats` - Maintenance statistics
- `GET /api/monitoring/services` - Service health monitoring
- `GET /api/monitoring/metrics` - Performance metrics
- `GET /api/audit` - Audit trail logging

#### 2. Start Frontend Application
```bash
# In a new terminal, navigate to frontend directory
cd frontend
npm run dev
```

Frontend will be available at: **http://localhost:3000**
